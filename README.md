# MobileLabsRN2025

Цей репозиторій містить реалізацію трьох лабораторних робіт з курсу мобільної розробки з використанням React Native.

## Технології
- React Native
- Expo
- React Navigation
- styled-components
- react-native-gesture-handler
- TypeScript
- FlatList, ScrollView та інші стандартні компоненти

---

## Лабораторна робота №1

**Тема:** Використання Expo для створення найпростішого додатку React Native.

### Основні кроки:
- Налаштоване середовище розробки для React Native.
- Реалізовано найпростіший застосунок з базовими екранами.
- Навігація реалізована через `@react-navigation/native`.
- Відображення простих елементів інтерфейсу, стилізація за допомогою базового inline-стилю або `StyleSheet`.

### Результати:
- Підключено емулятор Android та фізичний пристрій через Expo.
- Продемонстровано базове використання компонентів: `View`, `Text`, `Button`, `Image`.

---

## Лабораторна робота №2

**Тема:** Стилізація компонентів у React Native, списки, темізація.

### Реалізовано:
- Застосунок у стилі Steam.
- Використання `styled-components` для стилізації.
- Підтримка темної та світлої тем.
- Створено наступні екрани:
  - **Store:** список ігор
  - **Community:** стрічка новин
  - **Chat:** список чатів
  - **Safety:** налаштування безпеки
  - **Профіль користувача**

### Додатково:
- FlatList з підтримкою infinite scroll.
- Компоненти винесені у папку `components/`.
- Дотримано модульності.
- Коміти у форматі `тип: опис` (наприклад, `feat: add GameCard`).

---

## Лабораторна робота №3

**Тема:** Використання жестів у React Native за допомогою `react-native-gesture-handler`.

### Реалізовано:
- Гра-клікер з різними типами жестів:
  - `TapGestureHandler` – одиночний та подвійний тап
  - `LongPressGestureHandler` – утримання
  - `PanGestureHandler` – перетягування
  - `FlingGestureHandler` – свайпи вліво/вправо
  - `PinchGestureHandler` – масштабування

- Екран із завданнями, які оновлюються при виконанні:
  - Зробити 10 кліків
  - 5 подвійних кліків
  - Утримання об'єкта 3 секунди
  - Перетягування
  - Свайпи
  - Масштабування
  - Набір 100 очок

### Додатково:
- FlatList для відображення списку завдань зі статусом виконання.
- Реалізовано навігацію між екранами через `@react-navigation/native`.
- Компоненти винесено у модульні файли.
- Дотримано вимог до коду та оформлення.
